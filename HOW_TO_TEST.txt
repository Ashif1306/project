⚠️  TROUBLESHOOTING

MASALAH: Server tidak start
SOLUSI:
  cd kaloriz
  python manage.py runserver

MASALAH: Port 8000 sudah dipakai
SOLUSI:
  python manage.py runserver 8001
  Akses: http://127.0.0.1:8001/

MASALAH: Static files tidak load
SOLUSI:
  python manage.py collectstatic

MASALAH: Database error
SOLUSI:
  python manage.py migrate

MASALAH: Tidak bisa login admin
SOLUSI:
  Username: admin
  Password: admin123
  (case sensitive!)
